# 🧮 7-Function Calculator

A versatile command-line calculator supporting:

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧮 Modulus
- ⏫ Exponentiation
- 📐 Tangent calculation (uses `math.tan()`)

## Features
- Clean, user-guided menu
- Input validation
- Math-powered with Python’s `math` module

## Run the Program
```bash
python "7-Function Calculator.py"
